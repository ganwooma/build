HOW TO SERVER SETUP

이 문서는 didi's 다크 포레스트 리터치 버전의 서버 셋업을 도와주는 문서입니다.

1. JDK 설치

본 플러그인은 JDK 21을 지원합니다.

▶ Oracle JDK 21 설치

리눅스에서는 다음 명령어로 OpenJDK 21을 설치할 수 있습니다:

sudo apt update

sudo apt install openjdk-21-jdk

PaperMC 1.20.4는 JDK 22를 지원합니다.

▶ Oracle JDK 22 설치

리눅스에서 JDK 22 설치:

wget https://download.oracle.com/java/22/latest/jdk-22_linux-x64_bin.deb

sudo apt install ./jdk-22_linux-x64_bin.deb

자신의 운영 체제에 맞는 버전을 설치하세요.

2. 플러그인 빌드

깃허브에서 플러그인 소스를 다운로드한 뒤, src/main/resources/config.yml을 설정합니다.

이후 터미널에서 해당 디렉토리로 이동하여 다음 명령어를 실행합니다:

윈도우: gradlew shadowJar

리눅스: ./gradlew shadowJar

생성된 JAR 파일은 build/libs/에 있으며, 서버의 plugins 폴더에 넣으십시오.

무기 강화 플러그인, Simple Voice Chat 플러그인도 함께 추가해 주세요.

💡 참고: 빌드 과정은 윈도우 환경에서 진행하는 것을 권장드립니다.

3. 서버 파일

▶ 서버 파일 다운로드 (SETUP.zip)

압축을 해제한 후, 운영 체제에 따라 다음 파일을 실행합니다:

  • 윈도우: start.bat
  • 리눅스: start.sh

처음 실행 전, eula.txt에서 eula=false를 eula=true로 변경하십시오.

그 후, plugins 폴더에 플러그인들을 넣고 서버를 다시 실행하십시오.
