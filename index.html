HOW TO SERVER SETUP

이 문서는 didi's 다크 포레스트 리터치 버전의 서버 셋업을 도와주는 문서입니다.

1. JDK 설치

본 플러그인은 JDK 21을 지원합니다.

Oracle JDK 21 설치를 권장드립니다.

Oracle JDK 21 설치

리눅스에 경우

sudo wget https://download.java.net/java/GA/jdk21/fd2272bbf8e04c3dbaee13770090416c/35/GPL/openjdk-21_linux-x64_bin.tar.gz

를 사용하십시오

자신의 운영 체제에 맞는 버전을 다운로드 하시길 바랍니다.

2. 플러그인 빌드

Intellij를 설치하십시오.

Intellij 설치

Intellij에서 Minecraft Development Plugin을 설치하십시오.

깃허브에서 설치한 파일을 프로젝트 파일로 오픈하십시오.

src/main/resource/config.yml을 설정하고 터미널에서 ./gradlew shadowJar를 실행하십시오.

프로젝트 파일/build/libs/에 jar파일이 있을 것입니다.

이 파일을 서버 plugins 파일에 넣으십시오.

마찬가지로 무기 강화 플러그인, Simple Voice Chat 플러그인도 넣으십시오.

💡 참고: 빌드를 위해선 윈도우를 사용하시는 것을 추천드립니다.

3. 서버 파일

서버 파일 다운로드

에서 SETUP.zip을 다운로드 받고 압축을 푸십시오

윈도우에선 start.bat, 리눅스에서 start.sh를 실행하십시오.

eula.txt파일의 eula=false를 eula=true로 변경하고 다시 실행하십시오

plugins 폴더에 plugin 파일을 넣고 다시 start.bat / start.sh를 실행하십시오.
